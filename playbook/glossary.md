# glossary.md — 専門用語・略語（「それ知らない」を無くす）

> この事業・サイトの言葉を定義。Claudeが前提知識として参照する。

## 業態・口コミの用語
| 用語 | 意味 |
|---|---|
| メンエス | メンズエステ。男性向けのリラクゼーション/オイルマッサージ業態 |
| セラピスト（キャスト） | 施術を行うスタッフ。サイトの主役コンテンツ |
| 出勤（スケジュール） | セラピストのシフト/出勤予定 |
| 施術 | マッサージ等のサービス内容 |
| コース | 時間・料金のメニュー（例: 60分/90分） |
| 指名 | 特定セラピストを選んで予約すること |
| 体入（体験入店） | 新人セラピストの試用出勤 |
| og:image | 店舗サムネに使うOGP画像（店舗登録時に必ず設定） |

## サイト/プロジェクトの用語
| 用語 | 意味 |
|---|---|
| メンエスマップ | 本サイト（全国のメンエス店舗・セラピスト・口コミのポータル） |
| W2R（Write-to-Read） | 口コミを書くと他人の口コミが読める仕組み |
| owner_manual | 運営が代筆/手動投稿した口コミの`user_id`（即公開扱い） |
| is_public | 口コミの公開フラグ（1件目は自動公開トリガー） |
| 閲覧権（クレジット） | 口コミ読み放題の日数（700字以上投稿で自動付与） |
| group_id / brand_XXX | 同一ブランドの複数店舗を束ねるグループID（brand_+ハッシュ） |
| detailed_ratings | 6軸評価 {cleanliness, looks, style, service, massage, intimacy} |
| タグ白リスト | UIに存在する20タグ（体型6/雰囲気5/年代5/属性4）。これ以外禁止 |
| フライホイール | 口コミ→索引→検索流入→投稿者→口コミ、の自己強化ループ |
| 機会スコア | 表示 ×（3位想定CTR − 現順位CTR）。口コミ投入の優先度指標 |

## 技術用語
| 用語 | 意味 |
|---|---|
| GSC | Google Search Console（検索パフォーマンス・索引状況） |
| GA4 | Google Analytics 4（流入・行動・送客click_outbound） |
| CWV | Core Web Vitals（LCP/CLS/INP）。SEOの体験指標 |
| LCP / CLS | 最大コンテンツ描画 / 累積レイアウトシフト |
| SSR / ISR | サーバーサイド描画 / 増分静的再生成（トップはSSRに変更済み） |
| noindex,follow | このページは索引しないがリンクは辿る（空セラピストに付与） |
| canonical | 重複時の正規URL指定 |
| optimizeImageUrl | 外部画像をweserv.nl/Supabase/Unsplashで最適化する自作関数 |
| wsrv（images.weserv.nl） | 外部画像をリサイズ+WebP化する無料プロキシ |
| LazyImage | 遅延読込＋WebP変換の自作画像コンポーネント |
